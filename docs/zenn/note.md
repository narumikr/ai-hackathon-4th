# History Learning Agent

私たちが今回のAI Hackthonに向けて作成したものは、歴史学習のサポートおよび自分専用の歴史体験記作成エージェントです

## 背景・なぜ作ろうと思ったか

あなたは学生の頃、歴史の授業は面白かったでしょうか？それとも退屈だったでしょうか？私は退屈でした。周りの友達に聞いてみても歴史が好きな人は少数でした、一方歴史が好きな人たちのその大半が大河ドラマや歴史を題材にしたアニメやゲームの影響を受け、自ら学んだり、授業を通してしっかりと定着させていっていたような気がします

## プロジェクトの概要

### 対象ユーザー

- 学生
- 教育者
- 歴史好き
- 観光者

教育現場で学び、実際に課題に直面している学生やそんな生徒を指導している教育者の課題解決に向けてサポートをするだけではなく、元からの歴史好きやその地に足を踏み入れた観光客にとってもより良い体験を支援するプロダクトとなっています。

### 解決する課題

- 歴史教育の難解さ
  - 教科書の暗記が中心
  - 歴史に興味を持ちにくい
  - 現代とかけ離れていてイメージが湧かない
- 知識の断片化
  - 年号や事実の言葉による説明だけで臨場感がない
  - 教科書には必要最低限のことしか書かれない

他教科と違って歴史の授業はひたすら年号と出来事をインプットしていく、暗記が重要な科目となっていて、どうしても興味を持てないと授業が苦痛だったかと思います

加えて、衣食住の文化の違いや銃や刀を用いた戦さ、圧倒的な階級制度といった現代からあまりにもかけ離れた世界にイメージが全くわきません

他にも歴史というものは１冊の教科書に収まるわけもなく、学校では必要最低限のことしか教えられません

大河ドラマを見たり、歴史を題材にした漫画などに触れたときに「え？こんなことがあったの？！」という事実を知ったことはありませんか？

私たちはしょうがないとはいえ、断片化された知識だけを学んでおり、歴史を好きになれるきっかけを喪失しているのかもしれません

日本人にとって大事な文化、時代背景、思想など学校の授業で取りこぼしてしまっていた事柄を学ぶ機会をAIの力で生み出し、歴史は難しものというイメージを払拭する助けになれたらなとそう願います

### ソリューションの特徴

課題に対して大きく2つのアプローチをとっています

- 旅先への事前学習、情報収集
- 実際に触れた後の事後学習と定着

#### 旅先への事前学習、情報収集

ユーザーによって入力された行先をベースにAIが情報を収集

ユーザーはただ行先を入力するだけでAIが自動的に歴史的観点から関連するスポットを検索し、合わせて旅先として提案してくれます

またその際にはそれぞれのスポットに関する文献から取得された正確な情報の提供と提案内容に含まれている事柄の年表を表示してくれます

#### 実際に触れた後の事後学習と定着

AIとともに作成した計画で旅に繰り出し歴史に触れ終わった後に思い出の写真をアップロードして自分だけのパンフレットを作成します

ここでもAIが登場し、ユーザーインプットの感想とは他にアップロードされた画像を解析して補足情報も追加してくれます
