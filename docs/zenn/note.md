# History Learning Agent

私たちが今回のAI Hackthonに向けて作成したものは、歴史学習のサポートおよび自分専用の歴史体験記作成エージェントです

## プロジェクトの概要

### 対象ユーザー

- 学生
- 教育者
- 歴史好き
- 観光者

教育現場で学び、教示している学生や教育者、歴史好きやその地に足を踏み入れた観光客を対象としています

### 解決する課題

- 歴史教育の難解さ
  - 教科書の暗記が中心
  - 歴史に興味を持ちにくい
  - 現代とかけ離れていてイメージが湧かない
- 知識の断片化
  - 年号や事実の言葉による説明だけで臨場感がない
  - 教科書には必要最低限のことしか書かれない

どうしても他教科と違い暗記するだけ、現代とかけ離れすぎて想像すら困難な科目ゆえに敬遠されがちな歴史。

歴史の情報量は膨大で教科書では必要最低限のことしかかかれないことによる知識の断片化。歴史を好きになるきっかけの喪失。

そういった日本人にとって大事な文化、時代背景、思想などを学ぶ機会をAIの力で解決していけたらと思い、制作しました

### ソリューションの特徴

課題に対して大きく2つのアプローチをとっています

- 旅先への事前学習、情報収集
- 実際に触れた後の事後学習と定着

#### 旅先への事前学習、情報収集

ユーザーによって入力された行先をベースにAIが情報を収集

ユーザーはただ行先を入力するだけでAIが自動的に歴史的観点から関連するスポットを検索し、合わせて旅先として提案してくれます

またその際にはそれぞれのスポットに関する文献から取得された正確な情報の提供と提案内容に含まれている事柄の年表を表示してくれます
