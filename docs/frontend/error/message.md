
# フロントエンドのエラーメッセージ一覧

本ドキュメントは、バックエンド/API呼び出し失敗時にエラーダイアログへ表示される固定文言を、
「どこで」「どのタイミングで」「どのメッセージが出るか」で整理したものです。

## エラーダイアログ（API失敗時）

### 旅行一覧

- どこ: /travel（旅行一覧ページ）
- タイミング: 旅行一覧の取得に失敗したとき（listTravelPlans）
- メッセージ: 旅行一覧の取得に失敗しました。

### 旅行作成

- どこ: /travel/new（新規旅行作成ページ）
- タイミング: 旅行計画の作成に失敗したとき（createTravelPlan）
- メッセージ: 旅行計画の作成に失敗しました。

- どこ: /travel/new（新規旅行作成ページ）
- タイミング: 旅行ガイド生成に失敗したとき（generateTravelGuide）
- メッセージ: 旅行ガイドの生成に失敗しました。

### 旅行詳細

- どこ: /travel/[id]（旅行詳細ページ）
- タイミング: 旅行情報の取得に失敗したとき（getTravelPlan）
- メッセージ: 旅行情報の取得に失敗しました。

- どこ: /travel/[id]（旅行詳細ページ）
- タイミング: 旅行ガイド再生成に失敗したとき（generateTravelGuide）
- メッセージ: 旅行ガイドの生成に失敗しました。

- どこ: /travel/[id]（旅行詳細ページ）
- タイミング: 旅行計画の削除に失敗したとき（deleteTravelPlan）
- メッセージ: 旅行計画の削除に失敗しました。

- どこ: /travel/[id]（旅行詳細ページ）
- タイミング: 旅行完了処理に失敗したとき（updateTravelPlan: status completed）
- メッセージ: 旅行完了処理に失敗しました。

### 旅行編集

- どこ: /travel/[id]/edit（旅行編集ページ）
- タイミング: 旅行情報の取得に失敗したとき（getTravelPlan）
- メッセージ: 旅行情報の取得に失敗しました。

- どこ: /travel/[id]/edit（旅行編集ページ）
- タイミング: 旅行計画の更新に失敗したとき（updateTravelPlan）
- メッセージ: 旅行計画の更新に失敗しました。

- どこ: /travel/[id]/edit（旅行編集ページ）
- タイミング: 旅行ガイド再生成に失敗したとき（generateTravelGuide）
- メッセージ: 旅行ガイドの生成に失敗しました。

### 振り返り一覧

- どこ: /reflection（振り返り一覧ページ）
- タイミング: 振り返り一覧の取得に失敗したとき（listTravelPlans）
- メッセージ: 振り返り一覧の取得に失敗しました。

### 振り返り作成

- どこ: /reflection/[id]（振り返り作成ページ）
- タイミング: 振り返り用の旅行情報取得に失敗したとき（getTravelPlan）
- メッセージ: 振り返り用の旅行情報取得に失敗しました。

- どこ: /reflection/[id]（振り返り作成ページ）
- タイミング: 振り返りの作成に失敗したとき（uploadSpotReflection / createReflection）
- メッセージ: 振り返りの作成に失敗しました。

### 振り返り表示

- どこ: /reflection/[id]/view（振り返り表示ページ）
- タイミング: 振り返りの取得に失敗したとき（getTravelPlan）
- メッセージ: 振り返りの取得に失敗しました。

## 備考

- 上記メッセージは [frontend/src/constants/ui.ts](../../frontend/src/constants/ui.ts) の
	`ERROR_DIALOG_MESSAGES` で定義しています。
- エラーダイアログのタイトルは共通で「エラーが発生しました。」を使用します。
