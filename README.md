# 第4回 Agentic AI Hackathon with Google Cloud 提出作品

## プロジェクト名

歴史特化型旅行AIエージェント

## 一行説明

歴史学習を旅行体験と結びつけ、事前の学習から現地体験、振り返りまでを一貫して支援するAIエージェント

## 解決する課題

- 歴史は暗記中心になり、興味を持ちにくい
- 年号や事実が断片的になり、時代や場所のつながりを実感しにくい
- 現地体験と学習内容が分断され、記憶が定着しにくい

## 提案する解決策

事前学習、移動中の補助、体験後の振り返りを一貫して支援し、歴史を「調べる」だけでなく「体験して理解する」学習導線を提供します

## 主な機能

- 行き先と目標とするスポットの入力に基づいた、行き先の歴史年表とその他歴史的な名所の提案
- 行った先撮った写真や感想をアップロードすることで、旅行の振り返りをサポート

## 期待される効果

- 旅行と歴史の学習を結びつけ、興味関心を高める
- 体験と学習を結びつけ、旅行の経験を記録
